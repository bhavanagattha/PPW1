Sebentuk list unordered:

  • Kopi
  • Teh
  • Susu

Sebentuk list ordered:

  1. Kopi
  2. Teh
  3. Susu

Ordered list berfungsi untuk menunjukkan list yang memiliki urutan, sedangkan unordered list tidak.

ordered list juga memiliki beberapa tipe, yaitu tipe nuerik, alhabetik, dll. Tipe-tipe tersebut dapat digunakan sesuai kebutuhan. Contoh:

List bernomor:

  1. Apel
  2. Pisang
  3. Lemon
  4. Jeruk

List dengan huruf:

  1. Apel
  2. Pisang
  3. Lemon
  4. Jeruk

List dengan huruf kecil:

  1. Apel
  2. Pisang
  3. Lemon
  4. Jeruk

List dengan angka romawi:

  1. Apel
  2. Pisang
  3. Lemon
  4. Jeruk

List dengan angka romawi kecil:

  1. Apel
  2. Pisang
  3. Lemon
  4. Jeruk
